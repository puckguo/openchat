## Becoming a Contributor

We welcome contributions from everyone! Here's how you can contribute:

### Ways to Contribute

1. **Code**
   - Fix bugs
   - Add features
   - Improve performance
   - Write tests

2. **Documentation**
   - Improve guides
   - Translate documentation
   - Add examples
   - Fix typos

3. **Design**
   - UI/UX improvements
   - Logo and branding
   - Assets and graphics

4. **Community**
   - Help other users
   - Share your projects
   - Organize events
   - Spread the word

5. **Testing**
   - Report bugs
   - Test new features
   - Provide feedback

### Getting Started

1. Read [CONTRIBUTING.md](CONTRIBUTING.md)
2. Check [GitHub Issues](https://github.com/opencode-chat/opencode-chat/issues) for open tasks
3. Join our [Discord](https://discord.gg/opencode)
4. Start contributing!
