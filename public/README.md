# Open CoChat 客户端

OpenCode 多人协作聊天室前端客户端，支持实时聊天、AI 助手、文件分享等功能。

## 🚀 快速开始

### 方式一：直接打开使用（最简单）

1. 双击 `index.html` 文件用浏览器打开
2. 在"服务器地址"处填入你的服务器地址（如：`ws://192.168.1.100:3002`）
3. 输入会话名称（房间名）和用户名
4. 点击"连接"按钮

### 方式二：使用 HTTP 服务器（推荐）

如果需要文件上传功能，建议使用 HTTP 服务器：

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080

# 或使用 VS Code Live Server 插件
```

然后访问 `http://localhost:8080`

## 📋 使用说明

### 连接配置

| 字段 | 说明 | 示例 |
|------|------|------|
| 服务器地址 | WebSocket 服务器地址 | `ws://your-server-ip:3002` |
| 会话 ID | 房间名称，相同 ID 进入同一房间 | `team-chat` |
| 用户名 | 你的显示名称 | `张三` |
| 角色 | 用户权限 | `Member` |

### 功能介绍

**💬 实时聊天**
- 发送文字消息
- @提及用户（@ai @all）
- 查看在线成员

**🤖 AI 助手**
- 输入 `@ai` + 问题询问 AI
- 支持代码高亮和 Markdown 格式

**📎 文件分享**
- 点击"选择文件"按钮上传
- 支持图片、文档等多种格式
- 文件自动上传到云存储

**📝 对话总结**
- 点击"总结聊天"生成对话摘要
- AI 自动分析聊天内容

**🧹 清空记忆**
- 点击"清空AI记忆"重置上下文

## 🔧 故障排除

### 无法连接服务器

1. 检查服务器地址是否正确
2. 确认服务器防火墙已放行 3002 端口
3. 确认云服务器安全组允许 3002 端口

### 文件上传失败

1. 需要使用 HTTP 服务器方式打开（不能直接打开文件）
2. 检查 OSS 配置是否正确

### 页面显示异常

1. 使用 Chrome、Firefox、Edge 等现代浏览器
2. 清除浏览器缓存后重试
